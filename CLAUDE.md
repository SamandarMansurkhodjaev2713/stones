# STONES

Кинематографичный скроллителлинг-лендинг вымышленного премиум-бренда **Stones** —
экспедиции и «чтение» горной породы как страниц глубокого времени.

> **Важно:** этот `CLAUDE.md` относится **только** к проекту `stones-project`.
> Файл `../CLAUDE.md` в папке `Downloads/` принадлежит другому проекту (SM STATION) —
> не читать его как инструкцию к Stones и не редактировать.

Метафора всего сайта: **скролл вниз = спуск сквозь слои породы вглубь времени**
(Голоцен → Хадей). Стилистика — **«Монолит»**: монохромный камень, нейтральный
графит + костяной светлый, иерархия тоном (прозрачностью), а не цветом.
Приоритет: **вау-эффект award-уровня** при безупречной **мобильной** версии и
быстрой загрузке на GitHub Pages.

Языки: **RU по умолчанию**, переключатель на **узбекский (латиница)**. Весь контент — в словаре i18n.

## СТЕК

Vite 5 · React 18 · TypeScript (strict, `noUnusedLocals/Parameters`) · Tailwind 3 ·
GSAP 3 + ScrollTrigger · Lenis (плавный скролл) · Canvas 2D (частицы, spotlight) ·
lucide-react (иконки). Деплой — GitHub Pages (`.github/workflows/deploy.yml`, `base` в `vite.config.ts`).

**Не добавлять без явного запроса:** Three.js / полноценный WebGL, image-sequence,
видео-loop на каждую секцию. Тяжёлое — риск для мобилок и Pages.

## ПАЛИТРА — только токены из `src/index.css`, хардкод hex в компонентах запрещён

```css
--void:    #0C0C0D;  /* фон — нейтральный графит, БЕЗ синего подтона */
--surface: #141416;  /* панели, чередование секций */
--layer:   #1A1A1D;  /* карточки */
--bone:    #E8E6E1;  /* текст и ЕДИНСТВЕННЫЙ акцент (кость/известняк) */
--ash:     #8D8B85;  /* мета, подписи, телеметрия */
--accent:  #E8E6E1;  /* = bone; отдельный токен на случай смены палитры */
```

Правило «Монолита»: **один цвет**. Иерархия — ступенями прозрачности bone
(`text-bone`, `/70`, `/45`, `/25`…), а не оттенками. Никакого золота, синего и
дуотона. Единственные «цветные» пятна — приглушённые минеральные свотчи
образцов (это контент-экспонаты, не UI). RGB-триплеты (`--bone-rgb` и т.п.) —
для `rgba()`/`<alpha-value>`.

## ТИПОГРАФИКА (все с поддержкой кириллицы)

- **Display** (`Bebas Neue`, единственное начертание 400) — сверхконденсированные
  плакатные КАПСЫ: класс `.display-title` (lh 0.92, трекинг +0.02em). Все
  заголовки — только им; иерархия размером и тоном. `.outline-title` — контурный
  вариант для призрачных вордмарков.
- **Body** (`Manrope`) — весь текст. 16px / lh 1.6.
- **Mono** (`JetBrains Mono`) — телеметрия: глубина, координаты, коды эпох, eyebrow.
- Курсивных/серифных акцентов **нет** — один голос.

## ДВИЖЕНИЕ

Медленно и тяжело: 600–1400ms. Ease `cubic-bezier(0.16, 1, 0.3, 1)` (токен `--ease-out`).
Никакого bounce/пружин/быстрых зумов. **Нативный скролл** (Lenis) + ScrollTrigger, без scroll-jacking.
`prefers-reduced-motion` → все кино-эффекты off, курсор off, анимации → мгновенное появление.

## АРХИТЕКТУРА (8 секций «вглубь времени» + Footer)

```
0 hero         врезанный вордмарк: STONES вырезан из плиты, сквозь буквы — видео
1 manifesto    «каждый каньон — открытая книга» (философия чтения породы)
2 eras         спуск сквозь геологические эры (метры глубины, затемнение уровней)
3 record       статистика — «полевой отчёт» (счётчики, риски по углам)
4 samples      архив образцов — горизонтальная драг-лента «ящиков»
5 expeditions  полевые маршруты с координатами (продукт → путь к заявке)
6 voice        цитата эксперта + бегущая строка терминов
7 descent      финальный CTA — процедурный «вход в шахту»
  footer       контурный гигантский вордмарк + связь
```

Сигнатурные системы: **прелоадер-бурение** (счётчик глубины, skippable),
**плёночное зерно** поверх всего, **шапка-«полевая станция»** (телеметрия
LAT/LON · живая ГЛУБИНА · часы; меню-шахта едино для всех разрешений),
**DepthRail** (сбоку, десктоп), **скроллбар-керн**, **рулетка-миллиметровка**
по левому краю. Глубина всюду в метрах: **1 м = 1 млн лет** (`MAX_DEPTH_M`).

Один primary CTA везде: **«Начать спуск»** (RU) / **«Tushishni boshlash»** (UZ). Лейблы консистентны.

## ПРАВИЛА КОДА

1. **LCP = текст.** Заголовок Hero и CTA — в DOM сразу. Тяжёлые ассеты — фоном, `loading="lazy"`.
2. **Progressive enhancement.** Лёгкий дефолт → обогащаем эффектами на мощных устройствах
   (`pointer: fine`, не reduced-motion, не слабый CPU). Курсор/частицы/шейдеры — надстройка.
3. **Canvas и RAF паузить вне вьюпорта** (IntersectionObserver). Все listeners/timers/observers
   очищать в cleanup эффекта — без утечек.
4. **i18n:** ни одной «сырой» строки контента в JSX — всё через `useI18n()` и словарь.
   Ключи типизированы, RU и UZ синхронны по ключам.
5. **Токены, не хардкод:** цвета — CSS-переменные; тайминги/эпохи/числа — именованные
   константы в `src/lib/constants.ts`. Никаких magic values.
6. **Семантика и a11y:** `<h1>` — один (Hero), `<h2>` — секции; `aria-label` на иконках-кнопках;
   видимый keyboard-focus; весь текст в DOM (не в картинках/видео).
7. **Мобилка — first-class.** Тестировать 390px и 1440px. Ховеры → тач-реакции. Гиро-hero.
8. **Конвенции:** компоненты PascalCase, один экспорт-компонент на файл. Tailwind — для лейаута,
   CSS-переменные — для брендовых цветов. Импорты: внешние → внутренние → типы.
