# Арт-дирекция — Stones

Эстетика: **глубокое время, стратиграфия, полевая геология**. Кинематографично,
тяжело, сдержанно. Не неон, не «наука в лаборатории» — а обрыв породы на закате.

## Палитра — стратиграфический дуотон

Единственный источник — CSS-переменные в `src/index.css`. Хардкод hex в
компонентах запрещён.

| Токен | Hex | Назначение |
|---|---|---|
| `--void` | `#0A0D12` | фон |
| `--surface` | `#111621` | панели, чередование секций |
| `--layer` | `#161C28` | карточки |
| `--bone` | `#EDE6D8` | основной текст |
| `--ash` | `#8C8676` | мета, телеметрия, подписи |
| `--accent-warm` | `#D68A4E` | тёплый акцент — поверхность / песчаник |
| `--accent-cold` | `#5E86A6` | холодный акцент — глубина / сланец |
| `--accent` | — | **живой** акцент, warm→cold по `--depth` |

**Правило дуотона.** Драйвер в `src/lib/scroll.tsx` пишет `--depth` (0→1) и
интерполирует `--accent`/`--accent-rgb` от warm к cold на весь скролл. Компоненты
используют `var(--accent)` — цвет «дышит» глубиной сам, вручную его не задавать.

## Типографика (все с кириллицей)

- **Display** — `Unbounded` (600–800): монументальные заголовки секций.
- **Accent italic** — `Playfair Display` italic: поэтичные строки, скупо.
- **Body** — `Inter`: весь текст, 16px / lh 1.6.
- **Mono** — `JetBrains Mono`: телеметрия, глубина, коды эпох, eyebrow (13px,
  uppercase, ls 0.28em).

Утилиты: `.font-display`, `.font-accent`, `.font-mono-t`, `.eyebrow`, `.text-strata`
(градиент warm→live-accent для акцентных слов).

## Движение

Медленно и тяжело: 600–1400ms, ease `cubic-bezier(0.16, 1, 0.3, 1)` (`--ease-out`).
Никакого bounce/пружин/резких зумов. Нативный скролл (Lenis) + ScrollTrigger, без
scroll-jacking. Полное уважение `prefers-reduced-motion`. Подробнее — `animation-map.md`.

## Композиция и ритм

- Вертикальный ритм секций: `py-28 md:py-40`; контейнер `max-w-7xl px-5`.
- Чередование фонов `--void` / `--surface` для разделения глав.
- Асимметрия и наложения (Manifesto), редакторские строки-списки (Eras,
  Expeditions), сетка досье (Samples).
- Каждая секция открывается «seam» — тонкой линией разлома между слоями.
