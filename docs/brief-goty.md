# Бриф «Сайт года» — 56 утверждённых решений

Собран из 14 блоков вопросов. Это источник истины для двух этапов доводки.
Реализованные пункты помечаются `[x]`.

## Этап 1 — фундамент и главное

### Общесайт: палитра и материя
- [ ] **Магма-акцент + инверсия.** Тёплый акцент вводится ТОЛЬКО как свет
  (сердцевина луча, тление шва, hover CTA, Хадей); UI остаётся монохромным.
  Плюс одна инвертированная светлая секция (Voice) как ритм-удар.
- [ ] **Типографика: микс solid+outline в заголовках + вертикальные врезки**
  (названия глав боком на полях, параллакс).
- [ ] **Фоны: страты + световой след.** Дрейфующие линии-страты в каждой секции
  (геометрия зависит от глубины главы) + мягкий отклик фона на курсор.
- [ ] **Пасхалка: фонарь на пробел.** Зажатый Space (десктоп) гасит сайт,
  оставляя луч вокруг курсора.

### Общесайт: системы
- [ ] **Курсор v3:** drag-стрелки над лентой архива + настоящая лупа (×1.8)
  над фото-пластинами.
- [ ] **DepthRail → керн-разрез + живое число** глубины у каретки.
- [ ] **Два титра-акта:** полноэкранные перебивки перед Эрами («ВГЛУБЬ») и
  перед Descent («ДНО»).
- [ ] **Микро-звук:** каменные клики + гео-скрипы на границах секций
  (только при включённом тумблере звука).

### Хореография появлений
- [ ] Заголовки — посимвольный стэггер везде (единый язык с hero).
- [ ] Медиа — раскоп-маска снизу вверх + всплеск пыли по кромке.
- [ ] Списки — сначала чертится линия-разлиновка, затем текст из-под неё.
- [ ] Числа — набег + финальный «дожим стрелки прибора».
- [ ] CTA — лава-заливка снизу + текст-барабан при hover.
- [ ] Порядок — строгая цепь (шов → заголовок → подзаголовок → медиа →
  детали) + учёт направления скролла при повторных входах.
- [ ] Выход секций — расслоение (фон медленнее контента) + лёгкий blur
  (десктоп).
- [ ] Стыки — искра по шву + подсветка-щелчок числа глубины в шапке.

### Меню
- [ ] Открытие: плиты-разлом + страница позади тонет (scale 0.97, темнеет).
- [ ] Пункты: буквы становятся окном в фото секции + разрядка при hover.
- [ ] Фон плит: рельеф процедурных страт.
- [ ] Мобильное: те же плиты + фото-полоса-превью у каждого пункта.

### Hero
- [ ] Глубина: атмосферная дымка за камнем + двойной слой букв при движении.
- [ ] Вход: камень всплывает → луч зажигается → буквы встают.
- [ ] Идл: луч-автопилот при бездействии + искры с гребня.
- [ ] Выход: параллакс-прощание камня + сжатие луча в точку.

### Прелоадер
- [ ] Имена проходимых эр + строки телеметрии рядом со счётчиком.
- [ ] Звук бурения (при включённом звуке).
- [ ] Полный ритуал раз в сессию, дальше короткая версия.

## Этап 2 — секции, финал, мобилка

### Manifesto
- [ ] Буквица + фраза-выноска между абзацами.
- [ ] Пластины: линия-сканер при hover + локальный отклик слоёв.
- [ ] GhostEpoch: смена терминов по скроллу + отклик на курсор.
- [ ] Первый абзац читается скроллом слово за словом.

### Eras
- [ ] Смена эр: горизонт бурения + распад/сборка глифов.
- [ ] Фото: Кен-Бёрнс + параллакс от курсора.
- [ ] Рейл: мини-прогресс внутри эры + клики-навигация.
- [ ] Хадей: магма-разгорание + деликатная дрожь.

### Samples
- [ ] Лента: инерционный наклон карточек в движении.
- [ ] Карточка: 3D-выдвижение при hover + посимвольная печать шифра.
- [ ] Индикация: имя текущего образца + линейка с делениями.
- [ ] Финал: пустой ящик-CTA «Следующий образец — ваш».

### Expeditions / Stats / Voice
- [ ] Маршруты: фото-превью у курсора с инерцией + контурное имя.
- [ ] Досье: фото места + чертёж профиля с анимацией рисования линии.
- [ ] Stats → полноэкранный отчёт, цифры плакатного кегля, такт на показатель.
- [ ] Voice → светлая инвертированная секция + скраб-чтение цитаты.

### Descent / Footer
- [ ] Тоннель: фактура породы на рамках, магма в глубине, вход камеры +
  обратный отсчёт глубины до −4600 М (замыкает кольцо с прелоадером).
- [ ] Footer: дно шахты (−4600 М, контурный вордмарк за краем, плакатные
  ссылки с hover-превью, штамп «КОНЕЦ КЕРНА»).

### Мобильная версия
- [ ] Hero: долгое касание = фокус луча + двойной тап = вспышка.
- [ ] Гиро: везде (фото эр, карточки архива) + угол наклона в телеметрии.
- [ ] Вибрация: точечно (смена эры, снап карточки, открытие меню).
- [ ] Скролл: нативный на таче (Lenis только для мыши).
- [ ] Пины: и Эры, и Архив пинятся.
- [ ] Descent: полноэкранный колодец, CTA в зоне пальца.
- [ ] Hover-эффекты → авто-цикл по элементу в центре вьюпорта.
- [ ] Перф: авто-деградация по замеру fps (снимает blur/частицы/пыль).

### Контент и доступность
- [ ] Усилить 5–6 ключевых фраз + вся новая микрокопия (RU/UZ синхронно).
- [ ] Полный дубль для reduced-motion: статичные версии всех сцен, весь
  контент в DOM, focus-trap в меню, клавиатура везде.

### Рабочее соглашение
Спорные места (слабый эффект / вред перформансу) — останавливаюсь и спрашиваю.
