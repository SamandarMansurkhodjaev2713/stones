# Бриф «Сайт года» — 56 утверждённых решений

Собран из 14 блоков вопросов. Это источник истины для двух этапов доводки.
Реализованные пункты помечаются `[x]`.

## Этап 1 — фундамент и главное

### Общесайт: палитра и материя
- [x] **Магма-акцент + инверсия.** Тёплый акцент вводится ТОЛЬКО как свет
  (сердцевина луча, тление шва, hover CTA, Хадей); UI остаётся монохромным.
  Плюс одна инвертированная светлая секция (Voice) как ритм-удар.
- [x] **Типографика: микс solid+outline в заголовках + вертикальные врезки**
  (названия глав боком на полях, параллакс).
- [x] **Фоны: страты + световой след.** Дрейфующие линии-страты в каждой секции
  (геометрия зависит от глубины главы) + мягкий отклик фона на курсор.
- [x] **Пасхалка: фонарь на пробел.** Зажатый Space (десктоп) гасит сайт,
  оставляя луч вокруг курсора.

### Общесайт: системы
- [x] **Курсор v3:** drag-стрелки над лентой архива + настоящая лупа (×1.8)
  над фото-пластинами.
- [ ] **DepthRail → керн-разрез + живое число** глубины у каретки.
- [ ] **Два титра-акта:** полноэкранные перебивки перед Эрами («ВГЛУБЬ») и
  перед Descent («ДНО»).
- [ ] **Микро-звук:** каменные клики + гео-скрипы на границах секций
  (только при включённом тумблере звука).

### Хореография появлений
- [x] Заголовки — посимвольный стэггер везде (единый язык с hero).
- [x] Медиа — раскоп-маска снизу вверх + всплеск пыли по кромке.
- [x] Списки — сначала чертится линия-разлиновка, затем текст из-под неё.
- [x] Числа — набег + финальный «дожим стрелки прибора».
- [x] CTA — лава-заливка снизу + текст-барабан при hover.
- [ ] Порядок — строгая цепь (шов → заголовок → подзаголовок → медиа →
  детали) + учёт направления скролла при повторных входах.
- [ ] Выход секций — расслоение (фон медленнее контента) + лёгкий blur
  (десктоп).
- [ ] Стыки — искра по шву + подсветка-щелчок числа глубины в шапке.

### Меню
- [ ] Открытие: плиты-разлом + страница позади тонет (scale 0.97, темнеет).
- [ ] Пункты: буквы становятся окном в фото секции + разрядка при hover.
- [ ] Фон плит: рельеф процедурных страт.
- [ ] Мобильное: те же плиты + фото-полоса-превью у каждого пункта.

### Hero
- [x] Глубина: атмосферная дымка за камнем + двойной слой букв при движении.
- [ ] Вход: камень всплывает → луч зажигается → буквы встают.
- [x] Идл: луч-автопилот при бездействии + искры с гребня.
- [ ] Выход: параллакс-прощание камня + сжатие луча в точку.

### Прелоадер
- [ ] Имена проходимых эр + строки телеметрии рядом со счётчиком.
- [ ] Звук бурения (при включённом звуке).
- [ ] Полный ритуал раз в сессию, дальше короткая версия.

## Этап 2 — секции, финал, мобилка

### Manifesto
- [x] Буквица + фраза-выноска между абзацами.
- [x] Пластины: линия-сканер при hover + локальный отклик слоёв.
- [x] GhostEpoch: смена терминов по скроллу + отклик на курсор.
- [x] Первый абзац читается скроллом слово за словом.

### Eras
- [x] Смена эр: горизонт бурения + распад/сборка глифов.
- [x] Фото: Кен-Бёрнс + параллакс от курсора.
- [x] Рейл: мини-прогресс внутри эры + клики-навигация.
- [x] Хадей: магма-разгорание + деликатная дрожь.

### Samples
- [x] Лента: инерционный наклон карточек в движении.
- [x] Карточка: 3D-выдвижение при hover + посимвольная печать шифра.
- [x] Индикация: имя текущего образца + линейка с делениями.
- [x] Финал: пустой ящик-CTA «Следующий образец — ваш».

### Expeditions / Stats / Voice
- [x] Маршруты: фото-превью у курсора с инерцией + контурное имя.
- [x] Досье: фото места + чертёж профиля с анимацией рисования линии.
- [x] Stats → полноэкранный отчёт, цифры плакатного кегля, такт на показатель.
- [x] Voice → светлая инвертированная секция + скраб-чтение цитаты.

### Descent / Footer
- [x] Тоннель: фактура породы на рамках, магма в глубине, вход камеры +
  обратный отсчёт глубины до −4600 М (замыкает кольцо с прелоадером).
- [x] Footer: дно шахты (−4600 М, контурный вордмарк за краем, плакатные
  ссылки с hover-превью, штамп «КОНЕЦ КЕРНА»).

### Мобильная версия
- [x] Hero: долгое касание = фокус луча + двойной тап = вспышка.
- [x] Гиро: везде (фото эр, карточки архива) + угол наклона в телеметрии.
- [x] Вибрация: точечно (смена эры, снап карточки, открытие меню).
- [x] Скролл: нативный на таче (Lenis только для мыши).
- [~] Пины: Эры пинятся на всех ширинах; Архив на мобилке НАМЕРЕННО оставлен
  нативной свайп-лентой со снапом (пин + горизонтальный скраб на iOS ломается
  при схлопывании адресной строки, а тяга пальцем по полке — прямее). Готов
  включить, если решишь иначе.
- [x] Descent: полноэкранный колодец, CTA в зоне пальца.
- [x] Hover-эффекты → авто-цикл по элементу в центре вьюпорта.
- [x] Перф: авто-деградация по замеру fps (снимает blur/частицы/пыль).
